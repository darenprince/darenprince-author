Privacy Policy

Pic Detective uses OpenAI's image analysis API to process uploaded files. Images are sent securely to OpenAI for analysis and are not stored by Pic Detective beyond the time needed to complete the request.

According to OpenAI's policy, images submitted through the API may be retained for a short period to monitor abuse but are not used to build advertising profiles or shared with third parties.

By using this service you consent to this temporary processing. If you have concerns or wish to request deletion, contact support.
