# Crown Labs Bible Web Deployment

## GitHub Pages Deployment

1. Open repository settings.
2. Navigate to Pages.
3. Under Source select:
   - Deploy from a branch
4. Select branch:
   - crownlabsbible/foundation
5. Select folder:
   - /root
6. Save changes.
