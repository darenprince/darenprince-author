# Bookstore Logo and Badge Manifest

Last updated: 2026-07-01

This manifest tracks bookstore logos, retailer marks, and purchase badges available in the repo. Use it before adding or swapping retailer proof on the Game On landing page.

## Usage Rules

- Use `*-white*` assets only on dark backgrounds.
- Use `*-light*`, dark, blue, green, or black assets only on light backgrounds.
- Do not add strokes or bordered boxes around retailer logos unless the source asset already includes that treatment.
- Star ratings should remain yellow gold, not white or green.
- Keep Amazon first in purchase badge/logo groups unless a later test intentionally changes retailer priority.
- Use the horizontal Waterstones PNG for dark-background logo strips and modals where the square SVG reads too much like an icon.
